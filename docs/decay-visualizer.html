Shape Parameter Explained

        
How different shape values affect the decay curve. shape=1 is linear, >1 is slow start, <1 is fast start.

        

          
        

      

      

        
Practical Interpretation

        

          shape = 0.3 -- Aggressive early decay. "Yesterday matters, last week doesn't."

          shape = 0.5 -- Fast initial drop, long tail. Chat logs, ephemeral notes.

          shape = 1.0 -- Linear. Steady, predictable decline.

          shape = 1.5 -- Slow start, accelerates. "Still fresh for a while." Daily notes sweet spot.

          shape = 2.0 -- Quadratic. Stays very relevant, then drops off a cliff.

          shape = 3.0 -- Cubic. Almost no decay for a long time, then crashes. Reference docs.
