# Project Atlas

Atlas is a distributed data pipeline built by the Nexus Labs team. The project started in January 2025 and reached v1.0 in March 2026.

## Team
- **Sarah Chen** — Lead architect, designed the ingestion layer
- **Marcus Webb** — Backend engineer, built the transformation engine
- **Priya Patel** — DevOps, manages the Kubernetes deployment on AWS

## Tech Stack
- Rust for the core pipeline
- PostgreSQL for metadata storage
- Redis for job queuing
- Kubernetes on AWS EKS

Atlas processes approximately 2 million events per day for Meridian Corp, our primary client.
